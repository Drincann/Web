一、授课内容

  • HTML5 != H5

    H5涵盖的范围很大，它不是 HTML5、不是H5播放器。它是这些东西的父级，H5所指的就是移动网页本身，它能够包含所有这些分支。

    H5是整个概念的名词，或者是产品的名词，而HTML5是技术名词。


  • CSS3

  • JavaScript

    一种脚本语言。

    Node.js（后端开发）。

  • BOM&DOM

    文档和浏览器的对象模型，通过编程语言来操作。

  • XML

    用来结构化描述信息，存储信息，例如作为配置文件。

    DTD（Document Type Definition 文档类型定义）
    -XML文件的模板。
    -作为一种声明，定义XML文件中的元素、元素的属性、元素的排列方式、元素包含的内容等等。
    XSD (XML Schema Definition)
    -一种标准，一种标准，指出如何形式描述XML文档的元素。
    -属于XML Schema语言的一支。
